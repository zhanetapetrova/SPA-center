# SPA Center

A clean, multi-page spa website with a responsive layout, services, gallery, and a booking planner modal.

## Pages

- Home (index.html)
- Services (services.html)
- Gallery (gallery.html)
- About (about.html)
- Contact (contact.html)

## Features

- Responsive navigation with mobile menu
- Hero section with call-to-action buttons
- Booking planner modal (Mon–Fri, 09:00–16:30)
- Modern, warm spa-themed styling

## Usage

Open `index.html` in a browser. Use the “Book a Visit” button to open the appointment planner.

## Structure

- `index.html` — Home page
- `services.html` — Services list
- `gallery.html` — Image gallery
- `about.html` — About section
- `contact.html` — Contact info
- `styles.css` — Global styles
- `script.js` — Navigation + booking modal logic
- `img/` — Image assets
